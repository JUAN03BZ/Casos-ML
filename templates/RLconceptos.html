Conceptos básicos de Regresión Lineal

La regresión lineal es un modelo estadístico que permite predecir una variable dependiente a partir de una o más variables independientes. Es ampliamente utilizada en estadística y machine learning por su sencillez y efectividad.

Tipos de regresión Lineal:

  • Regresion lineal simple: se relaciona una única variable independiente con una variable dependiente. Por lo tanto, la ecuación de este tipo de modelo de regresión lineal es de la forma y=β0+β1x1.
  • Resgresion lineal múltiple: el modelo de regresión tiene varias variables explicativas y una variable respuesta. Por lo tanto, la ecuación de este tipo de modelo de regresión lineal es de la forma y=β0+β1x1+β2x2…+βmxm.

    • Supuestos principales:

    • Linealidad: La relación entre variables es aproximadamente lineal.
    • Independencia: Los errores son independientes entre sí.
    • Homoscedasticidad: Los errores tienen varianza constante.
    • Normalidad del error: Los errores siguen una distribución normal.
