Conceptos Básicos de AdaBoost

AdaBoost (Adaptive Boosting) es un algoritmo de aprendizaje supervisado que mejora la precisión combinando múltiples clasificadores débiles para crear un clasificador fuerte.

El método funciona entrenando iterativamente una secuencia de clasificadores simples, cada uno intentando corregir los errores de los anteriores al asignar mayor peso a los ejemplos mal clasificados. Al final, todos los clasificadores se combinan en una votación ponderada para predecir la clase.

AdaBoost es especialmente útil para problemas de clasificación binaria y es conocido por su capacidad para mejorar significativamente modelos que individualmente tienen un desempeño sólo ligeramente mejor que el azar.

Ventajas principales:

  • Fácil de implementar.
  • Mejora robusta en precisión con pocos clasificadores.
  • Se adapta bien a ruido y conjuntos de datos complejos.

Sin embargo, puede ser sensible a valores atípicos y ruido extremo si no se usa con cuidado.

Aplicación en este proyecto:

El algoritmo AdaBoost se usa para clasificar si un cliente quedó satisfecho o no con base en:

  • Tiempo de atención
  • Resolución del problema
  • Tono del agente
  • Canal de contacto

Estas variables permiten construir un modelo que predice el nivel de satisfacción con precisión.
