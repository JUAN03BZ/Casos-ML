Conceptos básicos de Regresión Logística

La Regresión Logística se utiliza para predecir resultados binarios como sí/no, verdadero/falso o compra/no compra.

En este proyecto:

  • Variable objetivo: ¿Compra el usuario? (1 = Sí, 0 = No)
  • Variables predictoras:
    • Tiempo en el sitio web
    • Número de clics en productos
    • Fuente de tráfico (orgánico, publicidad, redes sociales, email)
    • Nivel de ingresos (bajo, medio, alto)

Interpretación:

El modelo calcula la probabilidad de compra. - Si la probabilidad es baja → No compra - Si es intermedia → Posible compra - Si es alta → Compra segura
